What I do

I have more than 15 years' experience building software for clients all over the world. Below is a quick overview of my main technical skill sets and technologies I use. Want to find out more about my experience? Check out my online resume and project portfolio.

Technologies

Vanilla JavaScript

I leverage Vanilla JavaScript to build dynamic and interactive web applications, ensuring a strong foundation for more complex frameworks.

NEXT.js, React & Vue

These frameworks allow me to create high-performance, scalable web applications with a seamless user experience. I utilize NEXT.js for server-side rendering, React for building user interfaces, and Vue for its flexible and efficient design.

Node.js

With Node.js, I develop robust back-end services and APIs, enabling efficient handling of data-intensive applications and real-time functionalities.

Python & R-Project

Python is my go-to for data analysis, machine learning, and automation scripts, while R-Project is essential for statistical computing and data visualization tasks.

.NET 6-8

I use .NET for developing reliable and scalable enterprise-level applications, ensuring high performance and security.

npm, Gulp, Grunt & Yarn

These package managers and task runners streamline my development workflow, managing dependencies and automating repetitive tasks for increased productivity.

Docker, Kubernetes & KIND (Kubernetes in Docker)

I employ Docker for containerization, Kubernetes for orchestrating and managing containers at scale, and KIND for testing Kubernetes clusters in Docker, ensuring a consistent and efficient deployment process.

AWS

AWS provides a robust cloud infrastructure, which I utilize for deploying scalable and secure applications, leveraging services like EC2, S3, and Lambda to enhance functionality and performance.

Managment Tools, CI/CD, and Analytical Tools

Azure DevOps Work Items

Azure DevOps Work Items are utilized to track work and manage projects efficiently, providing a comprehensive platform for planning, collaborating, and delivering software solutions.

Octopus Deploy

Octopus Deploy facilitates my continuous delivery process by automating the deployment of applications and services across various environments, ensuring reliable and consistent releases.

Opentelemetry

Opentelemetry helps me monitor and troubleshoot applications by collecting, processing, and exporting telemetry data such as traces, metrics, and logs, thus improving observability for cloud-native software.

AWS Cloud Watch

AWS Cloud Watch allows me to monitor and manage AWS resources, applications, and services, providing actionable insights that help track performance, optimize resource usage, and swiftly respond to system-wide changes.

Grafana

Grafana enables me to create interactive dashboards that visualize and analyze data from various sources, helping to monitor system health and performance metrics effectively.
